# Linqly: Eternal Pulse ❤️

A hyper-animated, high-performance web experience themed around Love, personalized for Sneha.

## ✨ Features

- **Hyper-Scale Visuals**: Massive, shimmering typography and cinematic vignettes.
- **Lag-Free Engine**: Optimized canvas-based particle system running at 60fps.
- **Interactive Narrative**: A "Touch Heart" loop that intensifies with every interaction.
- **Cinematic Climax**: A massive particle explosion and transition after 10 interactions.
- **Dynamic Quotes**: A curated stream of romantic messages for Sneha.
- **Elegant Design**: Glassmorphism, velvet color palettes, and premium Playfair Display typography.

## 🚀 Getting Started

The project is built with pure HTML, CSS, and Vanilla JavaScript. No dependencies or builds required.

1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Experience the pulse.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **Vanilla CSS**: Premium styling and hardware-accelerated animations.
- **Vanilla JS**: High-performance canvas physics and state-based interaction logic.
